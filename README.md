# 信息敏感度日报 Skill

> 将信息敏感度训练框架转化为 AI Agent 每日自动执行的轻量级情报采集流程。4 层并行采集，约 5-8 分钟生成一份 HTML 日报，适合 10 分钟快速浏览。

## 是什么？

每天自动从公开互联网采集、交叉验证、结构化汇总信息，生成**信息敏感度日报**——帮你比大多数人更早察觉政策、行业、资本、风险的风向变化。

## 覆盖领域

可在首次设置时选择（可多选）：

| 领域 | 内容 |
|------|--------|
| 金融财经投资 | 股市/基金/证券、宏观政策、央行/财政部公告 |
| 房产地产 | 限购/信贷、土拍、住建部政策 |
| 跨境电商/外贸 | 关税/海外政策、出海品牌、物流法规 |
| 新兴小众行业 | 宠物经济/汉服/3D打印/二次元等 |
| AI/科技 | 大模型、AI应用、融资、监管动态 |

## 特色功能

**📊 招聘信息 + 技能趋势（可选）**

每日采集 AI/Agent 岗位，含大厂动态、热门岗位 TOP 5、高频技能词统计；开启周环比后还可对比上周技能趋势变化。支持同时采集**国企/事业单位招聘**（可选）。

**🧠 专业名词自动注记**

首次出现时附括号简要说明，零金融基础也能读。

**🎯 热搜自动过滤纯娱乐**

自动过滤明星八卦/综艺/体育，只保留经济/政策/科技/社会议题信号。

**🔧 首次使用引导配置**

询问关注领域、是否开启招聘信息，一次设置后每天一句话触发。

## 安装

**方式一：让 Agent 自动安装**

在 WorkBuddy 对话中粘贴本仓库链接，Agent 会自动下载并安装 Skill：

```
帮我安装这个 skill：https://github.com/dhx-nuist/info-sensitivity-daily-report
```

**方式二：手动安装**

```bash
# 克隆本仓库
git clone https://github.com/dhx-nuist/info-sensitivity-daily-report.git

# 将 SKILL.md 复制到 WorkBuddy skills 目录
cp info-sensitivity-daily-report/SKILL.md ~/.workbuddy/skills/info-sensitivity-daily-report/SKILL.md
```

## 使用方法

### 首次使用

在 WorkBuddy 对话中输入：

```
跑今日信息敏感度日报
```

Agent 会引导完成偏好设置（关注领域 → 招聘信息开关 → 是否定时），之后每天说这句话即可生成日报。

### 常用指令

| 指令 | 说明 |
|------|--------|
| `跑今日信息敏感度日报` | 执行当日采集 |
| `跑今日日报，特别关注：{主题}` | 加临时主题 |
| `汇总本周日报，提炼 3 个最强信号` | 周复盘 |
| `开启招聘信息` / `关闭招聘信息` | 动态切换招聘板块 |

## 日报结构

每日生成的 HTML 日报包含以下板块（可视配置调整）：

```
信息敏感度日报 · YYYY-MM-DD

0. 今日一句话（80字以内，机会🔥/风险🚨概括）
1. 政策与宏观（国内政策 + 房地产 + 外贸/关税 + 海外信号）
2. 热点与舆情（热搜TOP + 财经社区 + 镜像视角）
3. 行业与风险（AI/科技 + 房地产 + 跨境电商 + 风险案例）
4. AI/Agent 招聘（如开启：大厂动态 + TOP5岗位 + 技能趋势 + 国企/事业单位）
5. 信号汇总表（一表收尾，最多12条）
```

- 输出格式：**HTML**（带 CSS 样式，浏览器直接打开）
- 每条信息标注**来源**和**可信度**（A=权威 / B=行业媒体 / C=自媒体）
- 每个信号标注**强度**（★~★★★★★）和**类型**（机会/风险/观察）

## 红线与原则

1. **效率优先**：4 层并行采集，5-8 分钟出结果；拿不到的数据源直接跳过
2. **不写占位**：只展示实际获取的信息，未取到的板块不留痕迹
3. **注明来源**：每条信息附链接；无源不收录
4. **区分事实与判断**：事实标来源，判断写「agent 推测」
5. **警惕割韭菜**：金融直播/付费内容默认加 ⚠️ 标签
6. **控制篇幅**：300 行以内，10 分钟可读完
