# 信息敏感度日报 Skill

> 将知乎《训练信息获取敏感度的20种方法》转化为 AI Agent 每日自动执行的情报采集流程。

## 能做什么？

每天自动从公开互联网采集、交叉验证、结构化汇总信息，生成一份**信息敏感度日报**——帮你比大多数人更早察觉政策、行业、资本、骗局的风向变化。

**每日覆盖 5 大领域：**

| 领域 | 内容 |
|------|------|
| 金融财经投资 | 股市/基金/证券、宏观政策、雪球/天天基金热帖 |
| 房产地产 | 限购/信贷、土拍、政府工作报告与招标 |
| 跨境电商/外贸 | 关税/海外政策、出海品牌、物流法规 |
| 新兴小众行业 | 宠物殡葬/汉服/3D打印/二次元等千亿新赛道 |
| AI/agent/科技 | 大模型、AI应用、融资、监管动态 |

## 特色功能

**📊 招聘信息 + 技能趋势周环比（可选）**
每日采集 AI/Agent 岗位，并**与上周做技能关键词频率对比**——看哪些技能在升温、哪些在降温。可在首次设置中开启/关闭。

**📖 专业名词自动注记**
首次出现时附括号简要说明，零金融基础也能读。

**🎯 热搜自动过滤纯娱乐**
自动过滤明星八卦/综艺，只保留经济/政策/科技信号。

**🔧 首次使用引导配置**
询问你的关注领域、输出格式、是否开启招聘信息。

## 使用方法

### 安装 Skill

**方式一：从 Release 下载 zip**
1. 前往 [Release 页面](https://github.com/dhx-nuist/info-sensitivity-daily-report/releases) 下载最新 `info-sensitivity-daily-report.zip`
2. 在Codex或Claude Code中导入 zip 文件

**方式二：手动安装**
```bash
# 克隆本仓库
git clone https://github.com/dhx-nuist/info-sensitivity-daily-report.git

# 将 skill 文件夹复制到skills 目录

**方式三：自动安装**
将本项目的仓库链接复制在Codex或Claude Code等agent中让其自己安装

### 首次使用

在 WorkBuddy 对话中输入：

```
跑今日信息敏感度日报
```

Agent 会引导你完成偏好设置（关注领域、输出格式、招聘信息开关），之后每天说这句话即可生成日报。

### 常用指令

| 指令 | 说明 |
|------|------|
| `跑今日信息敏感度日报` | 执行当日采集 |
| `跑今日日报，特别关注：{主题}` | 加临时主题 |
| `汇总本周日报，提炼 3 个最强信号` | 周复盘 |
| `开启招聘信息` / `关闭招聘信息` | 动态切换招聘板块 |

## 日报示例

日报包含 8-11 个板块（视配置而定），结构化 Markdown 格式，便于检索和周复盘：

```
## 0. 今日一句话
> [今日最值得注意的信号]

## 1. 政策与宏观（早信号）
## 2. 资本与社区
## 3. 热点与舆情（已过滤娱乐）
## 4. 行业与新公司
## 5. 案例与风险（反诈+裁判文书）
## 6. 海外信号
## 7. 广告与流量
## 8. AI/Agent 招聘信息（可选）
## 9. 信号汇总表
## 10. 来源清单
```

## 红线与原则

1. **不涉灰色违法**：骗局案例仅作风险识别，不输出操作方法
2. **注明来源**：每条信息附链接；拿不到的直接跳过
3. **区分事实与判断**：事实标来源，判断标「agent 推测」
4. **警惕割韭菜**：金融直播/付费圈层默认标注风险提示
